# BeamRay Marketing Website

A beautiful one-page marketing website for BeamRay Marketing, specializing in local lead generation for home service businesses.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern Animations**: Smooth scrolling, hover effects, and scroll-based animations
- **Contact Form**: Functional contact form with validation
- **Professional Styling**: Clean, modern design with gradient backgrounds
- **SEO Optimized**: Proper heading structure and meta tags

## Upload Instructions for cPanel

### Method 1: File Manager (Recommended)
1. Log into your cPanel at hosting.com
2. Open **File Manager**
3. Navigate to the **public_html** folder (or your domain's folder)
4. Upload all files from this folder:
   - `index.html`
   - `styles.css`
   - `script.js`
5. Set file permissions to 644 if needed

### Method 2: FTP Upload
1. Use an FTP client (like FileZilla)
2. Connect to your hosting account
3. Navigate to the **public_html** directory
4. Upload all files from this folder

## Customization Notes

### Adding Your Photo
- Replace the placeholder in the founder section with your actual photo
- Name your photo file `kyle-photo.jpg` and upload it to the same directory
- Or update the `src` attribute in the HTML to match your photo filename

### Contact Information
Update the contact details in the Contact section:
- Email: Change `info@beamraymarketing.com` to your actual email
- Phone: Update `(555) 123-4567` to your real phone number
- Adjust business hours as needed

### Social Media
- Facebook link is already configured to: https://www.facebook.com/kylecampbell123/
- Add more social media links in the founder section if needed
